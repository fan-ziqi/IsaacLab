本地安装

本地安装#

小心

我们已停止支持 Isaac Sim 版本 4.2.0 及以下版本。我们建议使用最新的 Isaac Sim 5.0.0 发行版以获得最新的功能和改进。

有关更多信息，请参阅 Isaac Sim 发布说明 。

备注

我们建议系统要求至少具备 32GB RAM 和 16GB VRAM 以支持 Isaac Lab。对于启用渲染的工作流，可能需要额外的VRAM。有关 Isaac Sim 的完整系统要求列表，请参阅 Isaac Sim 系统要求 。

有关驱动程序要求的详细信息，请参阅 Technical Requirements 指南。

  • 查看 Linux故障排除 以解决Linux中的驱动程序安装问题

  • 如果您使用的是新的 GPU 或者遇到了当前驱动程序的问题，我们建议您从 Unix Driver Archive 上安装最新的 生产分支版本 驱动程序，使用 Linux 上的 .run 安装程序。

  • 建议在升级到 Ubuntu 22.04.5 kernel 6.8.0-48-generic 或更新版本时使用 NVIDIA 驱动程序版本 535.216.01 或更高版本。

Isaac Lab 是建立在 Isaac Sim 平台之上的。因此，在使用 Isaac Lab 之前，必须先安装 Isaac Sim。

Isaac Sim 和 Isaac Lab 都提供了两种安装方式: 通过二进制下载/源文件，或者通过 Python 的包管理器 pip 。

安装方法可能取决于使用场景以及用户所需的自定义程度。例如，从 pip 安装 Isaac Sim 将比从二进制文件安装更为简单，但源代码将仅通过已安装的源代码包访问，而不能通过直接下载二进制文件访问。

同样，通过 pip 安装 Isaac Lab 仅推荐用于使用 Isaac Lab 以外的外部启动脚本的工作流程。Isaac Lab 的 pip 包仅提供 Isaac Lab 的核心框架扩展，并不包括任何独立的训练、推理和示例脚本。因此，推荐将此工作流程用于作为 Isaac Lab 外部扩展构建的项目，这些项目使用用户定义的运行脚本。

我们建议使用 Isaac Sim pip 安装来获得简化安装体验。

对于刚开始使用 Isaac Lab 的用户，我们建议通过克隆仓库来安装 Isaac Lab。
